# Execute e implante seu aplicativo AI Studio

Isso contém tudo o que você precisa para executar seu aplicativo localmente.

Visualize seu aplicativo no AI Studio: <https://ai.studio/apps/drive/1MnsRRyBfHsdBEv4820puEZEn8y928rVc>

## Executar Localmente

**Pré-requisitos:** Node.js

1. Instale as dependências:
   `npm install`
2. Defina a `GEMINI_API_KEY` em [.env.local](.env.local) para sua chave de API do Gemini
3. Execute o aplicativo:
   `npm run dev`
